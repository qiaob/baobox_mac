Mac 工具集 界面设计稿

覆盖 M1 里程碑的六个核心界面：菜单栏入口、智能截图、标注编辑器、剪贴板历史面板、设置窗口与权限引导。整体遵循 macOS 原生设计语言（SF Pro 字体、系统材质、系统控件比例），品牌通过统一的青碧色贯穿。

设计基调

原生 App 的设计原则是"消失在系统里"：不自造控件，不与系统语言冲突；品牌只出现在选中态、主按钮和 App 图标上。

Screen 01 · 菜单栏入口

菜单栏常驻 + 工具列表菜单

App 唯一的常驻入口。菜单是一个纯工具列表：每个工具一行，悬停展开二级菜单（该工具的操作、历史、偏好设置）。菜单里不内嵌任何工具的具体内容——工具增长到十个、二十个，版式也不变。

  • 每个工具一行：主行只有图标、名称、主快捷键；新工具模块注册后自动追加一行，列表可滚动。
  • 二级菜单承载工具自身：操作（开始截图）、数据（历史、贴图管理）、该工具的设置入口，全部收在二级菜单里，主菜单永远干净。
  • 通用项沉底：设置、检查更新、退出与工具列表用分隔线隔开，位置固定。
  • 主行单击 = 默认动作：直接点"截图"行等于"开始截图"，不必进二级菜单。

Screen 02 · 截图

智能截图 Overlay（单快捷键）

只有一个截图快捷键 ⌘⇧2。唤起后自动检测意图：鼠标悬停即高亮识别到的窗口，点击 = 截取该窗口；按下拖拽 = 区域截图；⏎ = 当前屏幕全屏；Esc 取消。用户不需要事先选模式。

  • 零模式选择：一个快捷键覆盖三种截图。悬停时用系统窗口列表（CGWindowList）实时命中检测，被识别的窗口整体高亮并标注 App 名与尺寸。
  • 意图靠手势区分：单击（按下即抬起）截窗口；按下并移动超过阈值（约 4px）立即切入拖拽选区，松手前都可调整。
  • 选区可微调：八向手柄 + 方向键逐像素移动（⇧+方向键 ×10），右下角等宽数字实时显示尺寸，放大镜辅助像素级对齐。
  • 提示常驻：底部提示条列出当前状态下的全部按键动作，新手可读、熟手无感。

Screen 03 · 截图

标注编辑器（M2）

截图完成后可进入编辑器：顶部一排标注工具，画布中直接操作，底部是导出动作。默认工具为箭头——截图标注最高频的动作。

  • 工具即选即用：单字符图标 + 高亮选中态；颜色板五色，红色为标注默认色。

Screen 06 · 首次启动

权限引导

首次启动只做一件事：把两个系统权限讲清楚——为什么要、给不给会怎样。每项可单独跳转系统设置，也允许跳过（对应功能降级运行）。
